# 🎮 SFML2 Games Collection

## Overview

This repository contains a collection of games developed using **C++** and **SFML 2 (Simple and Fast Multimedia Library)**.

The purpose of this repository is to practice Object-Oriented Programming (OOP), game development concepts, collision detection, animations, audio effects, and event handling.

---

## Technologies Used

- C++
- SFML 2
- Object-Oriented Programming (OOP)

---

## Included Games

### 🐦 Flappy Bird
A classic Flappy Bird game with:
- Dynamic pipes
- Levels and increasing speed
- Particle effects
- Sound effects
- Score and best score system

---

### 🍎 Food Collector
A game where the player collects falling food items while avoiding losing lives.

Features:
- Multiple food types
- Increasing difficulty
- Particle effects
- Background music
- Score system

---

### ❌⭕ Tic-Tac-Toe
A two-player Tic-Tac-Toe game featuring:
- Win and draw detection
- Sound effects
- Restart functionality
- Simple graphical interface

---

### ⚽ Bouncing Ball
Arcade game with:
- Ball physics
- Dynamic speed levels
- Paddle shrinking system
- Glow and trail effects
- Game Over system

---

### 🚀 Chicken Invaders
Space shooter game consisting of **10 levels**.

Features:
- Spaceship movement and shooting
- Enemy chickens
- Score and lives system
- Progressive difficulty
- Boss battle at Level 5
- Final Boss at Level 10
- Sound effects and animations

---

## Skills Demonstrated

- Object-Oriented Programming
- Collision Detection

- Animation
- Audio Integration
- Particle Systems
- Game Physics
- User Interface Design
